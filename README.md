# How a Grandfather Clock Works

A scroll-driven 3D explainer about the chain of ideas inside a weight-driven pendulum clock: repeatable frequency, the pendulum period, escapement, gearing, and gravitational potential energy.

## Run locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm test
npm run build
```

The model is built from procedural Three.js geometry, so the project has no external 3D asset dependency.
